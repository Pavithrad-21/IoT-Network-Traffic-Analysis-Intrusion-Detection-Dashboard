# 🛡️ IoT Network Traffic Analysis & Intrusion Detection Dashboard


An interactive **Power BI dashboard** designed to analyze IoT network traffic, identify potential cyber threats, and provide actionable security insights using the **Bot-IoT dataset**.

---

## 📌 Project Overview

As IoT devices continue to grow in number, securing network infrastructure becomes increasingly important. This project focuses on analyzing large-scale IoT network traffic data to distinguish between normal and malicious activity, identify attack patterns, and visualize security risks through interactive dashboards.

The dashboard enables users to:

* Monitor network traffic behavior
* Compare attack and normal traffic activity
* Detect suspicious network events
* Identify targeted systems and malicious sources
* Analyze risk levels and attack trends

---

## 🎯 Objectives

* Analyze IoT network traffic patterns
* Identify intrusion attempts and malicious behavior
* Detect abnormal traffic spikes and high-risk events
* Build interactive cybersecurity dashboards
* Generate insights for network monitoring and threat detection

| Total Data Transferred     | 5+ Billion Bytes |

| Active Source Devices      | 21               |
| Active Destination Devices | 172              |
| High-Risk Traffic Share    | 52%              |

### Findings

* TCP was the most frequently used communication protocol.
* Attack traffic accounted for the vast majority of observed network activity.
* A small number of source IPs generated most malicious traffic.
* Certain destination systems were repeatedly targeted.
* More than half of all traffic was classified as high-risk.
* Attack activity showed noticeable time-based patterns.

## 📂 Dataset

### Bot-IoT Dataset

The dashboard is built using the **Bot-IoT Dataset**, a benchmark cybersecurity dataset developed by **UNSW Canberra Cyber** for IoT intrusion detection research.

**Dataset Source:**
https://research.unsw.edu.au/projects/bot-iot-dataset

### Dataset Features

* Source IP Address
* Destination IP Address
* Source Port
* Destination Port
* Network Protocol
* Packet Count
* Byte Count
* Service Information
* Timestamps
* Attack Labels

### Attack Categories

The dataset contains both normal traffic and multiple attack scenarios, including:

* DDoS Attacks
* DoS Attacks
* Reconnaissance Attacks
* Information Theft Activities

> ⚠️ Due to dataset size limitations, raw dataset files are not included in this repository. Please download the dataset directly from the official source.
